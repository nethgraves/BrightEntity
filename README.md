# BrightEntity

A Magisk/KernelSU module to fix the issue where brightness is too low when maximized on Transsion devices based on Unisoc.

## 📌 Overview

Some Unisoc devices has an issue where the maximum brightness level is artificially capped as level 255 on certain port ROMs, and Transsion (maybe others too) has the level higher than 255. This module restores the proper brightness level.

## ✅ Compatible Devices

- Transsion (Itel/Tecno/Infinix)


## 📦 Installation

1. Download the latest `BrightEntity.zip`
2. Flash via **Magisk** or **KernelSU**, **KernelSU Next**, **SukiSU Ultra**, **KittiSU**, **APatch/FolkPatch**
3. Reboot your device
4. Brightness will be back to normal as is!
